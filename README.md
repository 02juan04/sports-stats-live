# FootBall Hub

This is an app where you can view stats from many soccer/football leagues and teams

## Online Website (Vercel.app)

You can visit the online site [here](https://football-stats-live.vercel.app)

## Getting Started

- clone repo
- `npm install` and then `npm run dev`

## Features

- View Up to Date standings and fixtures for a variety of Leagues
- View Live Games, updated in Real Time
- Responsive Layout for Mobile and Desktop
- UI built with Tailwind CSS

## Stack Used

- Next.js
- Typescript
- External Football API
- Tailwind CSS
- Vercel for Deployment

## What I learned

- learned Next.js routing with the App Router
- Fetched and handled API data safely using TypeScript
- Learned the difference between Client and Server Components in Next.js
- Built a responsive, dynamic UI with conditional rendering
